# Programming-Synthesis

## How to run tis project

1. Create virtual environment

```
python3 -m venv venv
```

2. Activate the virtual environment
```
source venv/bin/activate
```

3. Install required libraries (right now dont need to, go to extras)
```
pip install -r requirements.txt
```
## Extras
1. Collect libraries in requirements.txt (after creating a venv)
```
pip install pipreqs
pipreqs ./
```



4. Check status and Add changes
```
git status
git add .
```


5. Commit changes
```
git commit -a -m 'some message'
```

6. Push changes
```
git push
```

7. Pull changes
```
git pull
```
